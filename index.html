This Website!

Preview View Code

A portfolio page made using vanilla Javascript and CSS and utilizing particlesJS

The journey of a thousand miles starts with a single step, or in this case a single page! This is my first attempt at a portfolio website and is relatively simple in design but gave me the chance to explore a few interesting bits of code.

The header uses intersection observer to create the same effect as bootstrap's scrollspy. The Home screen makes use of particlesJS to create the interesting particle effect in the background (try clicking it!)
